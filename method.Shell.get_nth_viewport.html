Description [src]

Gets a viewport from the shell given its index.

This is typically used along cog_shell_get_n_viewports() to iterate over the viewports:

CogShell *shell = get_shell();
for (gsize i = 0; i < cog_shell_get_n_viewports(shell); i++)
    handle_viewport(cog_shell_get_nth_viewport(shell, i));

Parameters

index gsize
 

Index of the viewport to get.

Return value

Returns: CogViewport
 

Viewport at the given index.
 The data is owned by the instance.
