Convenience base class for web views.

Most of the functionality dealing with web views when using the Cog core library uses CogView instead of WPEWebKit.WebView. The base class is extended to delegate the creation of its WPEWebKit.WebViewBackend to the current CogPlatform implementation. Optionally, platform plug-in implementations can provide their own view class implementation by overriding Cog.PlatformClass.get_view_type.

A number of utility functions are also provided.
