Description [src]

Get the type of the viewport implementation in use.

This function always returns a valid type. If the active CogPlatform does not provide a custom viewport type, the default built-in type included as part of the Cog core library is returned.

When creating viewports, this function must be used to retrieve the type to use, e.g.:

CogViewport *view = g_object_new(cog_viewport_get_impl_type(), NULL);

In most cases it should be possible to use the convenience function cog_viewport_new(), which uses this function internally.

Return value

Returns: GType
 

Type of the view implementation in use.
