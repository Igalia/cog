Description [src]

Sends a keyboard event to the web view.

Platform implementations must call this method instead of directly using wpe_view_backend_dispatch_keyboard_event() in order to give the embedding application the chance to handle keyboard bindings. See cog_view_set_use_key_bindings() for more details.
